\label{app:validacao}

Este apêndice agrupa material de validação que sustenta os Capítulos~\ref{ch:validacao} e~\ref{ch:resultados}. Nenhuma linha deste apêndice deve ser promovida a conclusão principal sem a classificação correspondente no corpo do relatório.
