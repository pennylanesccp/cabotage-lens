# Reference PDF Rename and Move Map

This file records the historical reference PDF organization originally
performed under the private `references/` folder for issue #9. After the
public-history cleanup, restored local reference files belong under
`docs/references/` and remain ignored by Git.

## Why this is safe

- Only PDF paths changed during the original organization; PDF contents were not edited.
- No PDFs were deleted by that organization step.
- `docs/references/core/Dados Relatorio 2.xlsx` is local-only and remains outside Git with the reference bundle.
- Classification is provisional and can be revised as the thesis framing settles.

The "Old path" column is historical. The "New path" column is the current
local-only restore location.
