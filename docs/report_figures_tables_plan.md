## Orientações Finais

- Não hardcodar numeração de figuras/tabelas no texto; usar `\ref{...}` no LaTeX.
- Não criar novos números para preencher figuras. Cada valor deve vir dos artefatos rastreados ou permanecer como mensagem qualitativa.
- No corpo principal, priorizar uma narrativa limpa: workflow, fontes, resumo de validação, comparação representativa e, se útil, decomposição de componentes.
- Deixar tabelas completas, campos de proveniência e deltas antigo--novo no apêndice ou em artefatos rastreados referenciados.
